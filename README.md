# AI-Generated UGC Ad Content Pack for Fitness Studio Gym

## Project Overview

This project demonstrates the use of AI and prompt engineering to create a complete User Generated Content (UGC) advertising content pack for Fitness Studio Gym.

The content was generated using structured prompts and designed to create engaging, conversion-focused marketing content suitable for social media platforms.
